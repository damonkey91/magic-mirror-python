# Magic-Mirror-Python

A simple Python-based GUI for a Magic Mirror Raspberry Pi project.

This is a very simple Magic Mirror GUI, written in Python for Raspberry Pi.


## Installations

```python
python3 -m pip install tk
```


## Run

```
export DISPLAY=:0
python3 main.py
```
